 Clash

配置文件
远程配置文件 http://{{ .domain }}/clash/config 一键导入
本地部署时配置文件 http://127.0.0.1:{{ .port }}/clash/config 一键导入
以上提供的配置文件本身不包含节点，而使用clash provider的方式加载远程节点。
Provider
所有节点 http://{{ .domain }}/clash/proxies
筛选节点示例 http://{{ .domain }}/clash/proxies?c=CN,HK,TW&speed=15,30&type=ss
❗️节点过多会造成软件卡顿，可以自行筛选节点后修改配置文件的proxy-provider部分。
Provider筛选参数

🌟 使用URL Query对Clash Provider进行筛选

Query Key Query Value 备注
类型 type ss,ssr,vmess,trojan 可同时选择多个类型
国家 c AT,CN,IN,HK,JP,NL,RU,SG,TW,US... 可同时选择多个国家
排除国家 nc AT,CN,IN,HK,JP,NL,RU,SG,TW,US... 可同时选择多个国家
速度 speed 任何数字 单个数字选择最低速度
两个数字选择速度区间
过滤 filter r,p,rp,nr,np,nrp 默认不进行过滤
r: 中转节点
p: pool节点
rp: 中转与pool节点
n*: 表示not
